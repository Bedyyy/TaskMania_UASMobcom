### Latar Belakang

**TaskMania** adalah aplikasi android yang saya buat guna menyelesaikan proyek akhir dan ujian akhir semester dari mata kuliah mobile computing. TaskMania adalah aplikasi notes, pengingat, atau TODO list dimana user bisa menginputkan apa yang ingin mereka lakukan atau hal penting yang ingin mereka simpan dalam bentuk teks. Alasan saya membuat aplikasi ini adalah saat mencari referensi terkait aplikasi sederhana yang memenuhi segala kriteria proyek akhir, aplikasi jenis ini adalah yang paling banyak referensinya dan merupakan *best practice* bagi pemula seperti saya. Aplikasi ini dibuat menggunakan bahasa pemrograman Kotlin dan menggunakan database Room, namun Room database merupakan sebuah sistem database yang dibuat berdasarkan SQLite atau secara tidak langsung merupakan versi simpel dari SQLite. Tentunya penggunaan dan sintaks kodenya juga hampir sama. Aplikasi ini juga mengandung sistem CRUD (Create, Read, Update, Delete).
